# Introduction to Python

## What is Python?

Execute the following code and see what happens. Feel free to change it and run it as many times as you want.

```
friends = ['Taylor', 'Alex', 'Pat', 'Eli']
for friend in friends:
    print("Hi " + friend)
```

Output:

```
Hi Taylor
Hi Alex
Hi Pat
Hi Eli
```
